# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this repository, please report it responsibly.

- Open a [GitHub Issue](https://github.com/MVulcansTrace/VulcansTrace-Windows/issues) and label it `security`, or
- Contact the maintainer directly through GitHub

Please include:

- A description of the vulnerability
- Steps to reproduce
- The affected component or file

## Scope

This policy applies to the source code in this repository only. VulcansTrace is a local analysis tool -- it does not transmit logs or data off the machine. Vulnerabilities in the detection logic, evidence packaging integrity, or input handling are in scope.

## Response

Reports will be acknowledged within 72 hours. Valid vulnerabilities will be patched and disclosed in a responsible timeline.
